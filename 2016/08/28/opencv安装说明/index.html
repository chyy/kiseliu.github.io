OpenCV安装说明

Posted on 2016-08-28   |   In Deep Learning   |  

说明：本文参考炼数成金讲师罗韵的课程《开源计算机视觉库OpenCV从入门到应用（第一期）》。

下载Linux版本源码

这里有两部分代码opencv和opencv_contrib需要下载，其中opencv库是opencv的核心库，opencv_contrib是最新贡献但不是很成熟的函数库。由于国内网络防火墙问题，在编译过程中，会因为ippicv_linux_20151201.tgz下载错误导致编译失败，因此事先也需要准备正确的压缩包。

安装第三方环境

安装源码前需要先安装好需要的第三方环境：

  • [compiler] sudo apt-get install build-essential
  • [required] sudo apt-get install cmake git libgtk2.0-dev pkg-config libavcodec-dev libavformat-dev
    libswscale-dev
  • [optional] sudo apt-get install python-dev python-numpy libtbb2 libtbb-dev libjpeg-dev libpng-dev libtiff-dev libjasper-dev libdc1394-22-dev

编译源码进行安装

Step1: 编译源码

在OpenCV目录下，创建build文件夹，然后进入build文件夹内部，使用cmake命令编译源码，这里我们也编译了opencv_contrib库。

mkdir build
cd build
cmake -D CMAKE_BUILD_TYPE=Release -D OPENCV_EXTRA_MODULES_PATH=/mypath/opencv_contrib/modules -D CMAKE_INSTALL_PREFIX=/usr/local ..

Step2: 替换被墙的压缩包

这里可能会遇到ippicv_linux_20151201.tgz库下载的问题，我们等编译完成之后，进入opencv/3rdparty/ippicv/downloads/linux-808b791a6eac9ed78d32a7666804320e/ 下，替换已有的错误的压缩包ippicv_linux_20151201.tgz，然后重新编译，当你看到下述提示即可。

Configuring done
Generating done

Step3: 编译安装

sudo make -j7 && make install

安装完成之后，我们可以使用下列命令查看具体安装了那些库，由于我们把opencv_contrib中的库也编译进来了，所以可以看到库非常多。

ll /usr/local/lib

然后为了方便后续对这些库的使用，下面还需要进行如下配置：

sudo vim /etc/ld.so.conf

然后添加 /usr/local/bin 到文件中，再执行更新命令：

sudo ldconfig

如果不添加这一行，在执行可执行文件的时候，可能会报错说不能找到某些扩展库。

测试安装是否成功

我们用下列 test_opencv.cpp 代码进行测试，该代码主要是读取图片，显示图片：

#include <iostream>
#include <opencv2/highgui/highgui.hpp>

int main(int argc,char* argv[]){
  const std::string window_name = "Cloud";
  const std::string input_pic = "Cloud.jpg";
  cv::Mat test_pic = cv::imread(input_pic);
  if(test_pic.empty()){
          std::cout<<"no input image"<<std::endl;
          return -1;
  }
  cv::namedWindow(window_name);
  cv::imshow(window_name,test_pic);
  cv::waitKey(0);
  return 0;
}

进行编译，链接该代码执行所需要的三个库：

g++ -o test_opencv test_opencv.cpp -lopencv_core -lopencv_highgui -lopencv_imgcodecs

然后执行编译好的文件./test_opencv，如果能够显示出你读取的文件，即说明安装成功。
